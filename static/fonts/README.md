# Fuentes Coolvetica y Kaoly

## Instrucciones para agregar las fuentes

1. **Descarga las fuentes:**
   - **Coolvetica:** Descarga desde [Creative Fabrica](https://www.creativefabrica.com/es/product/coolvetica/) o [FontsForYou](https://fontsforyou.com/es/fonts/52639-coolvetica.font)
   - **Kaoly:** Descarga desde [Creative Fabrica](https://www.creativefabrica.com/es/product/kaoly/)

2. **Convierte las fuentes a formato web:**
   - Usa una herramienta como [Transfonter](https://transfonter.org/) para convertir los archivos .ttf a formatos web (WOFF2, WOFF)
   - O simplemente copia los archivos .ttf directamente

3. **Coloca los archivos en esta carpeta:**
   - `Coolvetica-Regular.ttf` (o Coolvetica.ttf)
   - `Coolvetica-Bold.ttf` (si está disponible)
   - `Kaoly-Regular.ttf` (o Kaoly.ttf)
   - `Kaoly-Bold.ttf` (si está disponible)

4. **Formato de archivos soportados:**
   - .ttf (TrueType)
   - .woff (Web Open Font Format)
   - .woff2 (Web Open Font Format 2) - Recomendado

5. **Estructura recomendada:**
   ```
   static/fonts/
   ├── Coolvetica-Regular.ttf
   ├── Coolvetica-Bold.ttf
   ├── Kaoly-Regular.ttf
   └── Kaoly-Bold.ttf
   ```

Nota: Si no tienes las fuentes descargadas, el sistema usará fuentes alternativas similares.
